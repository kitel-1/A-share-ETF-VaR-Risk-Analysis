# A股ETF投资组合市场风险量化分析
**基于VaR、CVaR、蒙特卡洛模拟与历史情景压力测试的A股宽基ETF组合风险管理实战项目**

## 项目简介
本项目针对国内A股宽基ETF资产组合，完整实现金融行业三大主流市场风险计量模型，对比不同方法下的组合单日尾部风险，并结合**2021年后A股本土历史黑天鹅危机**开展极端压力测试。
全面覆盖「正常市场波动风险 + 极端崩盘黑天鹅风险」两大核心维度，输出可视化分布图、模型对比表格、最坏情景风险总结，完整还原机构基金风控分析的标准流程。

## 核心研究内容
1. **参数正态分布法 VaR / CVaR**
基于收益率正态分布假设，计算90%、95%、99%置信水平下的组合风险价值与尾部条件损失
2. **历史模拟法 VaR / CVaR**
不预设分布形态，直接使用真实历史行情排序测算尾部损失，贴合A股实际市场特征
3. **蒙特卡洛模拟 VaR**
通过乔列斯基分解还原多资产相关性，10000次行情模拟预测未来极端下行风险
4. **A股本土历史压力测试**
选取2021年后国内真实股市危机，测算极端行情下组合单日最大亏损与区间累计回撤
5. 收益分布可视化、多模型横向对比、最终风险汇总报告

## A股历史危机压力测试情景
- 2022年上海疫情A股系统性暴跌
- 2022年10月A股市场恐慌性下跌
- 2023年宏观经济预期调整下行行情
- 2024年10月A股整体性系统性回调

## 运行步骤
1. **数据准备**：将A股ETF日度价格CSV文件放入`data`文件夹，确保文件包含`Date`（日期）和各ETF代码列
2. **代码配置**：在`VaR_Market_Risk.ipynb`中修改`tickers`（ETF代码）、`weights`（组合权重）为自身数据
3. **运行代码**：按顺序执行Jupyter Notebook所有单元格
4. **查看结果**：自动生成三类VaR/CVaR数值、收益分布对比图、压力测试柱状图、风险汇总表

## 项目核心结论
1. A股ETF收益率显著偏离标准正态分布，具备**尖峰肥尾、左偏**特征，极端暴跌概率远高于正态分布预测
2. 历史模拟法受限于过往样本量，易低估日常尾部风险；蒙特卡洛模型因考虑相关性，风险结果更保守严谨
3. VaR仅能衡量常规市场下的大概率尾部风险，无法覆盖黑天鹅极端崩盘场景
4. 结合历史压力测试，才能全面评估投资组合在极端行情下的真实抗风险能力
5. 三种方法互补使用，构成完整的A股ETF组合市场风险评估体系
