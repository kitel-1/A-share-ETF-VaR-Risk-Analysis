# A股宽基ETF投资组合市场风险量化分析
**基于VaR、CVaR、蒙特卡洛模拟与历史情景压力测试的A股ETF组合风险管理实战项目**

---

## 项目简介
本项目针对A股宽基指数ETF投资组合，完整实现金融行业主流市场风险计量模型，覆盖**历史模拟法、参数正态法、正态分布蒙特卡洛模拟、对称学生t分布蒙特卡洛模拟**四大类方法，测算95%、99%置信水平下的组合VaR风险价值与CVaR条件风险价值。

同时结合**2021年后A股本土历史黑天鹅危机**开展极端压力测试，对比不同分布假设对A股收益率风险测算的影响，实证探究正态分布、对称厚尾t分布在A股市场的真实适配效果与模型固有缺陷，完整还原机构基金风控分析的标准流程，构建从“常规尾部风险”到“极端崩盘风险”的全维度评估体系。

---

## 数据选取
- **标的资产**：沪深300ETF、中证500ETF、创业板ETF、科创50ETF
- **组合方式**：四只ETF等权重构建分散化投资组合
- **数据类型**：日度收盘价 → 计算对数日收益率序列
- **核心研究指标**：组合收益率均值、波动率、VaR尾部风险价值、CVaR条件风险价值、极端情景最大回撤

---

## 核心研究方法
### 1. 历史模拟法
以历史真实损益分布作为风险基准，测算A股市场真实尾部下行风险，作为所有模型对比标杆，贴合A股实际市场特征。

### 2. 参数正态分布法 VaR / CVaR
基于收益率正态分布假设，计算90%、95%、99%置信水平下的组合风险价值与尾部条件损失，作为经典风控模型参考。

### 3. 正态分布蒙特卡洛模拟
假设资产收益服从标准正态分布，通过乔列斯基分解还原多资产相关性，开展10000次随机情景模拟未来损益分布，通过分位数计算组合VaR。

### 4. 对称学生t分布蒙特卡洛模拟
通过最大似然估计（MLE）拟合历史数据得到最优自由度`df`，利用金融经典尖峰厚尾t分布开展蒙特卡洛风险模拟，探究厚尾分布对A股极端风险的适配效果。

### 5. A股本土历史压力测试
选取2021年后国内真实股市危机，测算极端行情下组合单日最大亏损与区间累计回撤，覆盖VaR无法衡量的黑天鹅崩盘风险。
- 2022年上海疫情A股系统性暴跌
- 2022年10月A股市场恐慌性下跌
- 2023年宏观经济预期调整下行行情
- 2024年10月A股整体性系统性回调

## 实证结果（VaR测算对比）
| 测算方法               | 95% VaR | 99% VaR |
|------------------------|---------|---------|
| 历史模拟法             | 2.02%   | 3.24%   |
| 正态分布蒙特卡洛模拟   | 2.31%   | 3.31%   |
| 对称t分布蒙特卡洛模拟  | 3.03%   | 5.18%   |

---

## 核心结论
1.  A股宽基ETF收益率显著不符合标准正态分布，具备典型**尖峰厚尾、收益左偏**市场特征，极端暴跌概率远高于正态分布预测。
2.  正态分布模型本次数值贴合历史风险，但天生低估极端黑天鹅尾部崩盘风险，结果仅为样本巧合，无法反映A股真实厚尾特征。
3.  对称学生t分布左右尾部完全对称，无法匹配A股“慢涨快跌、暴跌远多于暴涨”的非对称行情，无论如何调整自由度，均会过度高估极端下行风险，A股实战测算效果劣于传统正态分布。
4.  历史模拟法受限于过往样本量，易低估日常尾部风险；蒙特卡洛模型因考虑资产联动相关性，风险结果更保守严谨。
5.  VaR仅能衡量常规市场下的大概率尾部风险，无法覆盖黑天鹅极端崩盘场景，必须结合历史压力测试，才能全面评估投资组合在极端行情下的真实抗风险能力。
6.  多模型互补使用，才能构成完整的A股ETF组合市场风险评估体系。

---

## 项目不足与未来优化方向
1.  仅使用对称型t分布，未采用更贴合A股非对称行情的偏态t分布模型。
2.  未考虑A股涨跌停制度、流动性约束等本土市场特殊规则，对极端行情的模拟仍有局限。
3.  后续可使用Cornish-Fisher高阶矩修正正态分布，大幅优化尾部风险拟合精度，更适配A股尖峰厚尾特征。

## 运行步骤
1.  **数据准备**：将A股ETF日度价格CSV文件放入`data`文件夹，确保文件包含`Date`（日期）和各ETF代码列。
2.  **代码配置**：在Jupyter文件中修改`tickers`（ETF代码）、`weights`（组合权重）为自身数据。
3.  **运行代码**：按顺序执行Jupyter Notebook所有单元格。
4.  **查看结果**：自动生成多模型VaR/CVaR数值、收益分布对比图、压力测试柱状图、风险汇总表。
